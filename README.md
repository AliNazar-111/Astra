# Astra - Offline Voice Assistant

A 100% offline, production-grade Windows voice assistant powered by local AI.

## Features
- Wake word detection ("Astra")
- Offline speech-to-text
- Local LLM for intent understanding
- OS-level automation (apps, keyboard, mouse)
- Browser automation
- WhatsApp Web automation
- Text-to-speech responses
- Whitelist-based safety system

## Installation
```bash
pip install -r requirements.txt
```

## Usage
```bash
python main.py
```

## Requirements
- Windows 10/11
- Python 3.9+
- 8GB RAM minimum (16GB recommended)
- Microphone and speakers
